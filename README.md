# School-projects
The programs in this repository are projects I did as part of a group or individually while working on obtaining my degree.

The Dungeon Crawler project is my attempt at a grid based dungeon crawler with a randomly spawned monsters and chest with a shop always in the center. It is written in Java.
The C-Tucker-Final-Project-V2 is a Trivia C# MVC app that calls an API for the questions and also displays a scoreBoard.
